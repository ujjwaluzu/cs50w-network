# Network

Network is a social media web application built with Django as part of Harvard's CS50 Web Programming with Python and JavaScript (CS50W).

Users can create posts, follow other users, like posts, and edit their own posts without reloading the page. The application also includes pagination and a personalized feed showing posts from followed users.

## Features

### Authentication
- User registration
- User login and logout

### Posts
- Create new posts
- View all posts
- Edit your own posts
- AJAX-powered post editing (no page reload)

### Profiles
- User profile pages
- View a user's posts
- View follower and following counts

### Following System
- Follow and unfollow users
- AJAX-powered follow/unfollow
- Dedicated Following feed

### Likes
- Like and unlike posts
 - AJAX-powered likes
 - Live like count updates without page refresh 
 
### Pagination 
 - Paginated post feeds 
 - Paginated profile pages 
 - Paginated following feed
